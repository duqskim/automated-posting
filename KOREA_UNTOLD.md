## Drama vs Reality — Shorts 별도 시리즈

3분 Shorts 시리즈. 드라마 장면 팩트체크 → 본편(EP N) 유입 CTA.

**구조:**
```
[00:00] 드라마 장면 설명 (대사/상황)
[00:20] "Here's what actually happened..."
[01:00] 역사적 팩트 + 시각 자료
[02:00] "The drama got [X] right / wrong because..."
[02:30] "Watch the full story → Episode N"
```

**예시 에피소드:**

| 드라마 | 팩트체크 주제 |
|--------|-------------|

| 기황후 | How powerful was Empress Ki really? |
| 킹덤 | Joseon's real disease outbreaks |

### History Mode — 강화된 팩트체크

```python
# 역사 콘텐츠는 strict 모드 적용
HistoryFactCheckConfig:
    mode = "strict"
    warn_on_uncertain = True       # 불확실한 것도 경고
    required_checks = ["dates", "names", "events", "statistics"]
    # 연도 ±5년 이내 오차도 disputed 처리
```
